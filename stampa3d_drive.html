Configurazione iniziale
Per sincronizzare con Google Drive serve un Client ID OAuth gratuito. Si fa una volta sola in 3 minuti.
1
Vai su console.cloud.google.com → crea un progetto (es. Stampa3D)
2
Menu → API e servizi → Libreria → cerca Google Drive API → Abilita
3
Credenziali → Crea credenziali → ID client OAuth → tipo Applicazione web
4
In Origini JavaScript autorizzate aggiungi http://localhost:8080. In URI di reindirizzamento autorizzati aggiungi http://localhost:8080/stampa3d_drive.html. Copia l'ID client.
5
Vai su Schermata consenso OAuth → aggiungi la tua email come utente di test
Incolla qui il tuo Client ID Google OAuth:

Il Client ID viene salvato solo su questo dispositivo (localStorage). Non viene mai inviato a nessun server.
