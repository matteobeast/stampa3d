Configurazione iniziale
Per sincronizzare con Google Drive serve un Client ID OAuth. Si fa una volta sola in 3 minuti.
1
Vai su console.cloud.google.com → crea un progetto (es. Stampa3D)
2
Menu → API e servizi → Libreria → cerca Google Drive API → Abilita
3
Credenziali → Crea credenziali → ID client OAuth → tipo Applicazione web
4
In Origini JS autorizzate aggiungi https://matteobeast.github.io. In URI reindirizzamento aggiungi https://matteobeast.github.io/stampa3d/stampa3d_drive.html
5
Vai su Schermata consenso OAuth → aggiungi la tua email come utente di test
